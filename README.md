# YouTube Video & MP3 Downloader

A modern desktop application built with Python and Tkinter for downloading YouTube videos and extracting audio as MP3.

## Features
- Download high-quality YouTube videos.
- Extract audio as high-quality MP3.
- Progress tracking with a real-time progress bar.
- Quality and format selection.
- Built-in FFmpeg support for seamless conversion.

## Installation

### Prerequisites
- Python 3.x
- FFmpeg (included in the `ffmpeg` folder for Windows users)

### Running from source
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Youtube-Downloader.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python youtube_downloader.py
   ```

## Building the Executable
Run the `build_exe.bat` file to generate a standalone executable using PyInstaller.
